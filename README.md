# julialaureau.com

Site personnel de Julia Laureau — Fondatrice, auteure, sonothérapeute.

## Stack

- Next.js 14 (App Router)
- React 18
- TypeScript
- CSS pur (pas de framework)
- Déploiement Vercel

## Démarrage local

```bash
npm install
npm run dev
```

Ouvrir [http://localhost:3000](http://localhost:3000)

## Déploiement

Le site est déployé automatiquement sur Vercel à chaque push sur la branche main.

Domain : julialaureau.com
